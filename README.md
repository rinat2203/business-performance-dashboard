# Business Performance Dashboard (Power BI)

Interactive Power BI dashboard for analyzing business performance, sales trends, and profitability.

## Tools Used
- Power BI
- Power Query
- DAX
- Data Modeling

## Dashboard Features
- KPI monitoring
- Profit analysis
- Sales segmentation
- Geographic sales analysis
- Time-based trend analysis
- Discount band analysis

## Skills Demonstrated
- Dashboard development
- KPI reporting
- Business intelligence analysis
- Data transformation
- Analytical storytelling
